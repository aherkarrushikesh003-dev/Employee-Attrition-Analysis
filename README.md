# Employee-Attrition-Analysis
Employee Attrition Analysis using Machine Learning (Random Forest) with Python, Pandas, Seaborn, and Scikit-Learn.
Model Performance:
- Algorithm: Random Forest Classifier
- Accuracy: 87.76%

Visualizations:
- Employee Attrition Count
- Department-wise Attrition
- Gender-wise Attrition
- Confusion Matrix
- Feature Importance Analysis
